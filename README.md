# dev-details
This Bash Script will tell you about your { Computer | Node | OS | Kernal | Desktop Env. } and many other usefull details 

# **INSTALLATION PROCESS**

    $ git clone https://github.com/kalmux1/dev-details.git
    $ cd dev-details
    $ chmod +x start.sh
    $ bash start.sh
